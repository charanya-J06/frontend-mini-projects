# Dark-Mode Toggle

## Overview
This project demonstrates a **clean implementation of light and dark mode** using **CSS variables** and **JavaScript**. Users can toggle between themes seamlessly using the button in the top-right corner.

The project emphasizes **dark mode functionality**, showing how to manage color schemes efficiently while keeping the layout professional and responsive.

---

## Features
- Toggle between **light and dark modes** with one button  
- Professional, clean, and readable design  
- Subtle card highlight to emphasize key content  
- Fully responsive layout for desktop and mobile  
- Uses **Font Awesome icons** for the toggle button  

---

## Technologies Used
- **HTML5**  
- **CSS3** (variables, responsive design)  
- **JavaScript** (DOM manipulation, class toggle)  
- **Font Awesome** (icons)
